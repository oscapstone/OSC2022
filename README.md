## How to build
* Compile bootloader.img and kernel8.img
```
make
```

### Kernel image
* Compile kernel8.img
```
make kernel8.img
```
* Run on qemu
```
make run
```
* Debug on qemu + gdb
```
make debug
```

### Bootloader image
* Compile bootloader.img
```
make bootloader.img
```
* Run on qemu
```
make load
```
